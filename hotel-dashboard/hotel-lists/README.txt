HOTEL LISTS
===========

This folder holds three files:

1. forbes-five-star-hotels.csv   — the Forbes Travel Guide five-star list
2. gold-list-hotels.csv          — the Condé Nast Traveller Gold List
3. master-hotels.csv             — the combined master list, with a "source"
                                   column showing which list(s) each hotel came from

The verified versions of these files live in the Cowork chat titled
"Condé Nast Gold List hotel database". To drop them in here, open that chat
and ask it to save the three files into this folder:
