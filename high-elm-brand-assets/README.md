# High Elm Studio — Brand Assets

This folder contains everything needed to keep work on-brand.

## Files

| File | What it is |
|---|---|
| `brand-guide.md` | Written brand guide — palette, typography, tone of voice, locked copy lines |
| `guidelines.html` | Visual brand guidelines page (open in any browser to view) |
| `logo-leaf-teal.svg` | Primary elm leaf mark — for light backgrounds (sand, stone, white) |
| `logo-leaf-reversed.svg` | Reversed elm leaf mark — for dark backgrounds (night, ink) |
| `wordmark-light-bg.svg` | Full wordmark "high elm studio." with leaf — for light backgrounds |
| `wordmark-dark-bg.svg` | Full wordmark "high elm studio." with leaf — for dark backgrounds |

## Quick reference

**Colours:** Sand `#F5F0E8` · Stone `#E0D8CC` · Mid `#C8BFB0` · Ocean `#5B9BAA` · Ocean Light `#8BBDC9` · Ocean Deep `#3D7A8A` · Storm `#243D4F` · Night `#111C26` · Ink `#0C1520`

**Fonts:** League Spartan 600 (headlines, UI, wordmark — always lowercase) · Cormorant Garamond Light Italic (display) · Raleway 200/300 (body)

**Wordmark rule:** always lowercase, always with the trailing period — "high elm studio."

**Voice:** calm, grounded, never corporate. We/our (never I). Never use "chatbot" — always "concierge system" or "AI system".
